# PPS-TASK-SEM2
This is TASK for the course PROGRAMMING FOR PROBLEM SOLVING.

Python Mini Projects Collection
Overview

This repository contains three Python-based mini projects designed to demonstrate the application of fundamental programming concepts in solving real-world problems. The projects focus on traffic management, interactive gaming, and student performance analysis.

Projects Included
1. Traffic Signal Control System

A system that adjusts green signal timing based on traffic density. It classifies traffic into levels and assigns appropriate signal durations to improve traffic flow.

2. Number Guessing Game

A console-based game where the user guesses a randomly generated number within limited attempts. It includes multiple difficulty levels and a scoring system.

3. Student Performance Analyzer

A program that analyzes student marks, calculates averages, and classifies students into categories such as Fail, Average, Good, and Excellent.
